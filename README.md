# python_exercises

## Description
This project contains Python exercises about working with files in Python.

## Topics
- Reading files
- Writing files
- Append mode
- Functions
- Loops
- Lists

## Exercises

### Exercise 1
Save product information into a file called products.txt.

### Exercise 2
Read products.txt and display products with prices higher than a given amount.

### Exercise 3
Read product information and calculate:
- Total number of products
- Sum of prices
- Average price
- Most expensive product
- Cheapest product

### Exercise 4
Create customer invoices and save purchase information into sales.txt.
